### Summary

You can set the file to be loaded on initial load.

- Key: `first-read`
- State: **Recommend**
- Initial state: "main.anov"

### Example

```package.json
{
  "first-read": "story/main.anov"
}
```

### Version Information

- Alice Novel: v0.9.0-rc1 or later
- Alice Console: *Unsupported*
