### Summary

You can set the game title.

- Key: `game-name`
- State: **Recommend**
- Initial state: "" (No Title)

### Example

```package.json
{
  "game-name": "Alice Wonder Land"
}
```

### Version Information

- Alice Novel: v0.9.0-rc1 or later
- Alice Console: *Unsupported*
