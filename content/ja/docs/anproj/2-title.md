### 説明

ゲームタイトルを設定可能です。

- キー: `game-name`
- 状態: **推奨**
- 初期設定: "" (タイトルなし)

### 例

```package.json
{
  "game-name": "Alice Wonder Land"
}
```

### 利用可能バージョン

- Alice Novel: v0.9.0-rc1 以降
- Alice Console: *未実装*
